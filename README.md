# Transcription/Captioning Web App

A web application for uploading video/audio files, transcribing them using OpenAI Whisper, and editing the captions.

## Structure

- **client/**: Next.js frontend application.
- **server/**: Node.js/Express backend application.

## Getting Started

### Prerequisites
- Node.js installed.
- OpenAI API Key.

### Server
1. Navigate to `server/`.
2. `npm install`
3. Create `.env` file with `OPENAI_API_KEY=your_key`.
4. `npm run dev`

### Client
1. Navigate to `client/`.
2. `npm install`
3. `npm run dev`
